# Botanica Plant Shop Analytics

Responsive React + Vite dashboard for plant shop operations. It includes live category and date filters, normalized mock data for plants, pots and plant-care products, configurable low-stock alerts, KPI summaries, Recharts revenue visualization, category performance, and a stock-prioritized product table.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
